# 简介
天气通APP是一款界面简洁，能够提供天气信息的类系统天气应用，具体内容有温度、风力、湿度、能见度、气压、天气状况等数据，此天气数据源采用心知天气API，可以获取到最近7天的天气信息，同时用户也能够通过输入想要查询的城市进行查询，智能的搜索功能能够获得国内的所有城市，然后通过精确定位获得目标城市的天气信息。
